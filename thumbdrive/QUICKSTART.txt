PLUG ME IN. Then in the VM terminal:

  cp -r /media/$USER/<USB_LABEL>/* ~/thumbdrive/
  cd ~/thumbdrive
  bash setup.sh

setup.sh will:
  1. Untar the repo to ~/ArtificiallyUnintelligent
  2. Copy K's best.pt weights into models/
  3. pip install --user from offline wheels (pymavlink, onnxruntime, etc)
  4. Print the next commands to type

Then YOU type these by hand:

  (Terminal 1)
    ~/start_px4.sh
    -> 1 (x500_vision)
    -> 1 (roboverse)
    -> 2 (skip QGC, fake-GCS handles it)
    Wait for `pxh>` prompt, then:
    commander set_ekf_origin 47.397742 8.545594 488.0

  (Terminal 2 — pick ONE)
    cd ~/ArtificiallyUnintelligent/searchctl

    # PRIMARY (proven 350 pts in sim earlier today):
    # K's wall-follow + FSM stuck-escape + bonus 4:20 hard-land + slow
    # gated periodic 360 scan + lower altitude + lower YOLO threshold.
    python3 controller.py --bonus --altitude 2.0 --conf 0.5

    # Fallback A — same as primary but higher altitude (3.0m default).
    # Try if 2.0m altitude is hitting low obstacles.
    python3 controller.py --bonus --conf 0.5

    # Fallback B — Hail-Mary: very low altitude + very aggressive YOLO.
    # Use if both primary + Fallback A failed to find yellow.
    python3 controller.py --bonus --altitude 1.5 --conf 0.35

    # Smoke test only (no detect / no map / no fake-GCS): just flight
    python3 controller.py --no-detect --no-map --no-fake-gcs

DETECTION MODEL: K's best.pt is the default (3 classes — yellow/red/toxic
barrels; toxic filtered out at output, label names remapped to underscores
to match org's example image: 'yellow_barrel' / 'red_barrel').
Default confidence threshold = 0.70 (K's tuning). Org confirmed no
penalty for incorrect detections, so feel free to drop --conf at venue.

Outputs land at: ~/ArtificiallyUnintelligent/searchctl/run_<ts>/
  STATUS.txt         <- live status: eligibility + bonus countdown + base score
  map.png            <- show the judge (now has detection markers!)
  detections/*.jpg   <- show the judge (bbox images)
  run_summary.json   <- has total flight time + unique counts + per-pose

After all runs, pick the best:
  cd ~/ArtificiallyUnintelligent/searchctl
  python3 slot_summary.py   <- estimates scores across all runs, shows winner

At-venue tuning (no code edits needed):
  --altitude 2.0    closer to floor — catches floor-level yellow targets
  --altitude 3.5    higher cruise if locker barrels look high
  --conf 0.5        lower confidence if K's model isn't firing
  --conf 0.35       even more aggressive (no penalty for false positives)
  --bonus           4:20 hard-land + early-exit on dual-colour detected

DISK HYGIENE (if you see "no space left" or VM is >90% full):
  rm -rf ~/PX4-Autopilot/build/px4_sitl_default/rootfs/log/*
  rm -rf ~/ArtificiallyUnintelligent/searchctl/run_*
  rm -f ~/ArtificiallyUnintelligent/searchctl/logs/run_*.log
  pip cache purge

IF SOMETHING BREAKS:
  - USB #1 broken -> try USB #2 (identical copy)
  - Drone won't arm -> did you set EKF origin? Re-do it in pxh> console
  - QGC needed -> our fake-GCS heartbeat should make PX4 happy without it
  - Camera missing -> python3 controller.py --no-detect (still flies + maps)
  - Map crash -> python3 controller.py --no-map (still flies + detects)
  - Drone stuck at a wall -> wait ~3s; FSM stuck-escape will kick in
                             (rotates 80° CCW in place, then re-acquires
                             wall — no translation, EKF-safe)
  - Everything else -> python3 controller.py --no-detect --no-map --no-fake-gcs
