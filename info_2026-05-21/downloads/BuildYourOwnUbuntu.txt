OPTION B. SETUP YOUR OWN UBUNTU ENVIRONMENT


Use the following steps to install the required libraries and tools onto your own Ubuntu 22.04 machine. These are only for Ubuntu 22.04 ubuntu. Use these steps after you have install Ubuntu 22.04.
Install development essentials
1.     sudo apt update
2.     sudo apt upgrade
3.     sudo apt install -y build-essential curl git wget software-properties-common
4.     sudo apt install -y python3-pip python3-venv
5.     sudo apt install python3-opencv
6. sudo apt install libgz-msgs10-dev 
7. sudo apt-get install python3-gz-transport13 python3-gz-msgs10
Install PX4 and MAVSDK
8.     cd ~
9.     git clone https://github.com/PX4/PX4-Autopilot.git --recursive
10.     bash ./PX4-Autopilot/Tools/setup/ubuntu.sh
11.  pip install mavsdk
12.  cd /tmp
13. wget  https://github.com/mavlink/MAVSDK/releases/download/v3.17.1/libmavsdk-dev_3.17.1_ubuntu22.04_amd64.deb
14.  sudo apt install libmavsdk-dev_3.17.1._ubuntu22.04_amd64.deb
15. sudo apt install libopencv-dev


Install Openvins
        15. sudo apt-get install libeigen3-dev libboost-all-dev libceres-dev libgz-transport13-dev libgz-msgs10-dev
        16. mkdir ~/tools
        17. cd ~/tools
        18. git clone https://github.com/rpng/open_vins/
19. cd open_vins/ov_msckf/
20. mkdir build && cd build
21. cmake -DENABLE_ROS=OFF ..
22. make -j4 
23. sudo make install
Setup the environment
24. Add the following lines to the end of ~/.bashrc


export PROTOCOL_BUFFERS_PYTHON_IMPLEMENTATION=python
export OGRE_RTT_MODE=Copy
export PX4_GZ_SIM_RENDER_ENGINE=ogre
export GZ_SIM_RENDER_ENGINE=ogre


25. sudo ln -s /usr/include/opencv4/opencv2/ /usr/include/opencv2/


26. sudo apt install ros-humble-ros-gzharmonic
